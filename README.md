## About the Project
This project demonstrates how to implement Google reCAPTCHA v2 in an HTML form with PHP backend validation. It includes:
- A responsive HTML form with modern UI design
- Two types of reCAPTCHA implementations: "I am not a robot" (checkbox) and invisible reCAPTCHA
- Google reCAPTCHA v2 integration for spam protection
- PHP backend verification of reCAPTCHA tokens
- Clear instructions for implementation

The project serves as a learning resource for developers looking to add bot protection to their web forms.
